## Proyecto para la ETCM

Prototipo de inteligencia artificial para la detección de entidades


## Comandos

Ejecutar aplicación
```
make run
```

Ejecutar pruebas
```
make test
```

Entrar
```
make console
```

Crear imagen
```
make install
```

Remover imagen
```
make remove
```

Mostrar contenedor cuando se esta ejecutando
```
make show
```

## Iniciar
```
make install
make test
make run
```
